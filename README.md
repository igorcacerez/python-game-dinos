# Jogo Pong Pong

Jogo simples feito em Python utilizando a biblioteca PyGame. 

# Para usar

Tendo o python instalado em seu computado, basta rodar o seguinte comando  **python main.py** em seu terminal, dentro da pasta do projeto.
